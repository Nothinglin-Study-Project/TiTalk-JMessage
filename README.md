## QiTalk

基于JMessage的Android聊天软件。
 
 
支持 单聊，群聊，和机器人(Ashe，基于图灵机器人）聊。
 
 
多种主题，多种语言。


apk下载地址：http://shouji.baidu.com/software/22110765.html
